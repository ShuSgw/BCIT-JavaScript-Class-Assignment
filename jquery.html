Main Selectors

jQuery has many selectors and you can control HTML elements with them. You do not need to type lots of syntax that Javascript needs to type. Also, actions such as event handlers, or effects stick to the selecter, such as $( "#box" ).css( "border", "3px solid red" );

Selectors for ID and Class

Instead of "document.getElementById" or "document.getElementByClassName", you can get HTML simply with jQuery selectors.

						
	<html>
		<body>
			<div id = "idBox" ></div>
			<div id = "classBox" ></div>
		</body>
	</html>

	<script>
		$(document).ready(function() {
			var $idBox = $( "#idBox" );
			var $classBox = $(".classBox");
		});

	</script>

Effects and Actions

When you need to separate tasks depending on some situations, the conditional structures should be used.There are two ways to create conditional structures in Javascript.

animate() Method

A condition you define is evaluated by “if” statement, and if a situation matches the condition, it executes a block you define. Also, you can create another different block from the true part. If the situation does not match the condition you define, “if else” statement executes another block as false. Moreover, you can add more conditions with “else if”.

						
	<script>
		
	$("button").click(function(){
		    $("div").animate({left: '250px'});
		}); 

	</script>

toggle() Method

A condition you define is evaluated by “if” statement, and if a situation matches the condition, it executes a block you define. Also, you can create another different block from the true part. If the situation does not match the condition you define, “if else” statement executes another block as false. Moreover, you can add more conditions with “else if”.

						
	<script>
		  $('#samplebtn_02').click(function() {
		    $('#sampleText').toggle("slow");
		  });
	</script>
